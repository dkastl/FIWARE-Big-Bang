<a name="configuration-parameters"></a>

## 構成パラメータ

`config.sh` ファイルを編集して構成を指定できます。

| 変数名        | 説明                                               | 既定値 |
| ------------- | -------------------------------------------------- | ------ |
| QUERYPROXY    | Queryproxy を使用するかどうか。(true または false) | false  |

<a name="how-to-setup"></a>

## 設定方法

```bash
QUERYPROXY=false
```
