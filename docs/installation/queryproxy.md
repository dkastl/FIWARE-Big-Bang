## Configuration parameters

You can specify configurations by editing the `config.sh` file.

| Variable name | Description                   | Default value |
| ------------- | ----------------------------- | ------------- |
| QUERYPROXY    | Use queryproxy. true or false | false         |

## How to setup

```bash
QUERYPROXY=false
```
